# 网络爬虫学习项目

## 📚 项目说明
这是一个**学习用途**的 Java 网络爬虫练习项目，用于理解 HTTP 请求、HTML 解析和数据提取的基本原理。

## ⚠️ 免责声明
1. 本项目仅用于**学习和研究目的**
2. 请遵守目标网站的 `robots.txt` 协议和服务条款
3. 请勿将本项目用于商业用途或大规模数据采集
4. 使用者需自行承担法律责任

## 🛠️ 技术栈
- Java
- Jsoup (HTML 解析)
- Apache HttpClient (HTTP 请求)

## 📖 学习内容
- URL 提取和去重
- 相对路径处理
- HTTP 请求模拟
- HTML 结构解析

## 🚀 运行方式
bash mvn spring-boot:run
